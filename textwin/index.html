TextWIn - Texting Without Internet

Um Sistema de Bate-Papo Para Demonstração de Um Novo Método de Comunicação Móvel

Download do App Android



Objetivos

O objetivo deste projeto é a criação de um novo método de comunicação móvel, que permite a troca de dados (arquivos, mensagens de texto) mesmo em falta ou inoperância da rede convencional (Internet, rede de dados celular, e também sem Wi-Fi local).


Justificativa e Vantagens

Independência e/ou ampliação dos serviços de comunicações (Internet, redes de dados celular).

Redução dos custos com serviços de comunicações (Internet, redes de dados celular).

Possibilidade de comunicação em desastres naturais (ex. terremoto). Internet e rede de dados celular fica inoperante


Funcionamento

O método passa por três fases de comunicação para que a mensagem seja entregue ao dispositivo destino.

Fase 1

Funcionamento Normal com Internet/Servidor

Fase 2

Funcionamento apenas com Rede Wi-Fi Local (Roteador)

Fase 3

Funcionamento apenas com Rede Wi-Fi Local (Dispositivos)

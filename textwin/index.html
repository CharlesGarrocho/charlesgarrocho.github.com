TextWIn - Texting Without Internet

Um Sistema de Bate-Papo Para Demonstração de Um Novo Método de Comunicação Móvel

Download do App Android



Objetivos

O objetivo deste projeto é a criação de um novo método de comunicação móvel, que permite a troca de dados (arquivos, mensagens de texto) mesmo em falta ou inoperância da rede convencional (Internet, rede de dados celular, e também o Wi-Fi local).


Justificativa e Vantagens

Independência, ampliação e redução de custos dos serviços convencionais de comunicações (ex. Internet, redes de dados celular, etc).

Possibilidade de comunicação em desastres naturais (ex. em um terremoto a Internet e rede de dados celular podem ficar inoperante).

Uma pessoa com problemas de locomoção poderá se comunicar com outras pessoas na proximidade mesmo sem Internet, Rede de dados Celular ou Wi-Fi Local.


Funcionamento

O método passa por três fases de comunicação para que a mensagem seja entregue ao dispositivo destino.

Fase 1

Funcionamento Normal com Internet/Servidor

Fase 2

Funcionamento apenas com Rede Wi-Fi Local (Roteador)

Fase 3

Funcionamento apenas com Rede Wi-Fi Local (Dispositivos)
